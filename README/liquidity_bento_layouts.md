## Typography Hierarchy:

### **Liquidity USD (Primary):**
- `text-5xl font-bold text-green-400`
- `bg-gradient-to-r from-green-400 to-emerald-400`
- `text-shadow-lg shadow-green-500/50`

### **Price & Volume (Secondary):**
- `text-3xl font-semibold text-white`
- `text-shadow-md`

### **Change & Stats (Tertiary):**
- `text-xl font-medium text-slate-300`
- `text-shadow-sm`

### **Labels:**
- `text-sm text-slate-500 uppercase tracking-wide`

## Glass-Morphism Elements:
- `backdrop-blur-sm bg-slate-800/30`
- `border border-slate-700/50`
- `shadow-2xl shadow-slate-900/50`
- `hover:shadow-green-500/20 transition-all duration-300`

## Expand Functionality:
- **All layouts**: Expand button loads DEXScreener chart
- **Smooth transitions**: `transition-all duration-300`
- **Chart integration**: Direct API call to DEXScreener
- **Responsive design**: Adapts to different screen sizes
